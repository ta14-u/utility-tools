# .cursor ディレクトリ

このディレクトリには、Cursor エディタ用のプロジェクト固有の設定とルールが含まれています。

## ファイル構成

- `rules`: プロジェクトのコーディング規約、開発ガイドライン、技術スタック情報
- `settings.json`: プロジェクト固有の設定（パッケージマネージャー、ビルドツールなど）
- `README.md`: このファイル（ディレクトリの説明）

## 使用方法

これらのファイルは Cursor エディタが自動的に参照し、AI アシスタントの提案やコード生成時に考慮されます。

プロジェクトの構造や技術スタックが変更された場合は、これらのファイルを適宜更新してください。
