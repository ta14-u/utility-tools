# Utility Tools

自作した便利ツールをまとめたリポジトリです。GitHub Pagesで公開しています。

👉 [https://ta14-u.github.io/utility-tools/](https://ta14-u.github.io/utility-tools/)

## 機能

- **QR Code Generator**: テキスト入力からQRコードを生成
- 今後思いついたツールを追加していく予定

## インストール

1. リポジトリをクローン
2. 依存関係のインストール：

   ```bash
   pnpm install
   ```

## 開発方法

開発サーバーを起動：

```bash
pnpm dev
```
